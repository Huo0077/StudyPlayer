# 🎬 极简视频播放器 (Video Player) v8

这是一个基于 PyQt5 和 VLC 核心开发的高级视频播放器，支持笔记、分段标记和断点续播。

## 主要功能
- **全格式支持**：基于 VLC 核心，支持几乎所有视频格式。
- **高级笔记系统**：支持画笔、荧光笔、文字笔记，支持撤销/重做和自动保存。
- **分段标记**：支持对视频进行分段并保存标记。
- **记忆播放**：自动记录每个视频的最后播放位置。
- **汇总视图**：一键生成所有笔记的汇总大图（长图）。

## 🚀 快速启动
1. 直接运行安装程序进行安装。
2. 双击桌面快捷方式或文件夹内的 **`极简视频播放器.exe`** 即可使用。

## 📂 笔记与导出位置
为了方便管理，所有的笔记和截图都存放在程序所在目录的以下文件夹中：
- **`notes_aggregated/`**：生成的汇总笔记长图 (PNG)。
- **`snapshots/`**：视频截图 (PNG)。
- **`metadata/`**：分段信息与手写笔记的原始数据 (JSON)。
- **`progress.txt`**：视频播放历史记录。

## 快捷键
- **Space**: 播放/暂停
- **← / →**: 后退/快进 10s (长按右键可 3 倍速播放)
- **↑ / ↓**: 增减音量
- **Ctrl + Z / Y**: 笔记撤销/重做
- **Ctrl + S**: 手动保存当前数据
- **F / F11**: 全屏切换
- **S**: 视频截图

## 优势
- **极简设计**：拟态风格界面，自动隐藏控制栏，沉浸式观影体验。
- **完全绿色**：不产生系统垃圾，所有数据均保存在本地文件夹。
